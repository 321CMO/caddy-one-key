Caddy-Manager 一键管理脚本
这是一个为 Linux 服务器设计的 Caddy 自动化运维工具。通过简单的菜单交互，您可以快速完成 Caddy 的安装、反向代理配置、备份及日常维护。

🚀 快速开始
在您的服务器终端输入以下命令即可运行（支持 Debian/Ubuntu）：

Bash

wget -O caddy.sh https://raw.githubusercontent.com/321CMO/caddy-one-key/main/caddy.sh && chmod +x caddy.sh && ./caddy.sh
注意：首次运行并选择“安装”后，脚本会自动注册系统命令。此后，您在任何目录下只需输入 caddy 即可调出管理菜单。

✨ 核心功能
一键安装/卸载：基于官方仓库安装最新稳定版，自动配置 systemd 服务。

智能反向代理：

支持快捷添加域名转发。

语法预检：修改配置前自动校验，若有语法错误则自动回滚，确保服务不宕机。

自动格式化：自动运行 caddy fmt 保持 Caddyfile 整洁。

自动化唤醒：安装后，脚本会自动关联 caddy 命令，无需记忆长路径。

配置安全：

提供一键备份功能，记录每次修改的时间戳。

卸载脚本前强制二次确认，防止误操作。

运维助手：集成实时日志查看、站点列表显示、防火墙端口自动放行（UFW/Firewalld）。

⚠️ 注意事项
环境限制：本脚本目前主要针对使用 apt 包管理器的系统（如 Debian, Ubuntu, Linux Mint）。

域名解析：在使用反向代理功能前，请务必确保您的域名已解析到当前服务器 IP，否则 Caddy 无法成功申请 SSL 证书。

权限要求：脚本涉及系统服务管理，必须以 root 权限运行。
